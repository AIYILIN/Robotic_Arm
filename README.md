# Smart_Robo_Arm(智能桌面机械臂)

这是一个基于小米cybergear关节电机、unitree宇树关节电机、foc闭环步进电机的智能桌面机械臂。

## 主要涉及：
1.机械臂相关正逆运动学解算，2.机械部分设计，3.rs485，fdcan通讯，4.ps2遥控，5、freertos

## 硬件:

1、stm32H723

2、电源分电板

...
